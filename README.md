
# Fitness app 
## Features: 
- Edit Goals (Keep progress of your day with progress bar!)
  - Calorie Goal
  - Protein Goal (g) 
  - Carbs Goal (g) 
- Input meals with: 
  - Meal Type (Breakfast, Lunch , Dinner, Snack)
  - Calories 
  - Protein (g)
  - Carbs (g)
  - Sugar (g)

- Delete Meals
